# 🔐 Vaulty – Secure Password Manager

Vaulty is a modern, secure, and lightweight password manager designed to help users safely store, manage, and organize their credentials in one place. Built with a clean UI and strong encryption, Vaulty ensures your sensitive data always stays protected on your device.

## ✨ Features

- 🔒 Secure password storage with encryption  
- 📱 Clean and modern Flutter UI  
- 🔍 Fast search for saved credentials  
- 📋 One-tap copy username & password  
- 🏷️ Organize accounts by categories  
- 🔑 Biometric authentication (Fingerprint / Face Unlock)  
- ⚡ Lightweight and fast performance  
- 💾 Local secure storage on device  

---

## 🛡️ Security

Vaulty is designed with privacy-first principles:

- AES-level encryption for sensitive data  
- All data stored locally on your device  
- Biometric lock protection  
- No third-party data sharing  
- Fully offline-first secure storage  

Your data remains fully under your control at all times.
